# PaySmart Backend

Simple Express + MongoDB backend for PaySmart.

## Setup

1. Install dependencies:
   ```bash
   cd backend
   npm install
   ```

2. Start the server:
   ```bash
   npm run dev
   ```

3. API endpoints:
   - `POST /api/auth/register`
   - `POST /api/auth/login`
   - `GET /api/users/profile`
   - `PUT /api/users/profile`
   - `GET /api/expenses`
   - `POST /api/expenses`
   - `GET /api/payments`
   - `POST /api/payments`
   - `GET /api/wallets`
   - `POST /api/wallets`
